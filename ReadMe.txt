========================================================================
    CONSOLE APPLICATION : PT2INET Project Overview
========================================================================

This is a console application to convert Cube Voyager PT transit
LIN files to INET transit files (used in TranPlan transportation 
modeling software).

Programmed in C++ using Microsoft Visual C++ Express Edition

Compile and run without options for program arguments.

Author: Andrew Rohne - arohne@oki.org - www.oki.org
